# Projeto TED N1 - Algorítmos e Estrutura de Dados I

Este projeto foi desenvolvido como atividade avaliativa (TED N1) da disciplina de Algorítmos e Estrutura de Dados I, com o objetivo de aplicar conceitos fundamentais de lógica e desenvolvimento em C#.

## Sobre o Projeto

O sistema consiste em um menu interativo no console que reúne 8 funcionalidades diferentes, cada uma representando um exercício prático de programação.

## Funcionalidades

1. **Calculadora de Média Ponderada**
   Calcula a média de três notas com pesos diferentes.

2. **Classificador de IMC**
   Calcula o IMC e classifica o resultado.

3. **Calculadora Simples**
   Realiza operações básicas: soma, subtração, multiplicação e divisão.

4. **Gerador de Tabuada**
   Exibe a tabuada de um número informado pelo usuário.

5. **Somador com Condição de Parada**
   Soma números positivos até que um número negativo seja digitado.

6. **Sistema de Validação de Senha**
   Solicita a senha até que o usuário acerte.

7. **Conversor de Temperaturas**
   Converte entre Celsius e Fahrenheit utilizando funções.

8. **Mini Simulador de Caixa Eletrônico**
   Permite depósitos, saques e consulta de saldo com controle de estado global.

## Conceitos Aplicados

* Variáveis e Tipos de Dados
* Operadores Matemáticos e Lógicos
* Estruturas Condicionais (`if`, `else`, `switch`)
* Estruturas de Repetição (`for`, `while`, `do-while`)
* Métodos com e sem retorno
* Escopo de variáveis (local e global)

## Tecnologias Utilizadas

* C#
* .NET (Console Application)
* Microsoft Visual Studio Community

## Observações

Este projeto foi desenvolvido com fins educacionais, focando na prática dos conceitos básicos de programação estruturada.
